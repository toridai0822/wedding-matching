# ダウンロード手順

## このプラットフォームからファイルを保存する方法

### 方法A: 手動でダウンロード
1. 左側の「ファイルエクスプローラー」タブをクリック
2. 各ファイルを開いて、内容をコピー
3. パソコンのメモ帳に貼り付けて保存

### 保存するファイル一覧
```
wedding-matching/
├── index.html
├── admin.html
├── README.md
├── css/
│   └── style.css
└── js/
    ├── participant.js
    ├── matching.js
    └── admin.js
```
