# ステップ2: GitHubアカウントを作る（10分）

GitHubは「ファイルを保管する場所」です。無料で使えます。

## 2-1. GitHubのサイトに行く

🌐 https://github.com にアクセス

## 2-2. アカウントを作る

1. 右上の「Sign up」（登録）をクリック
2. メールアドレスを入力
3. パスワードを決める（覚えやすいもの）
4. ユーザー名を決める（例: taro-yamada）
5. メールが届くので、確認する

**完了！** ログインできるようになりました。

---

## 2-3. 新しいリポジトリを作る

「リポジトリ」= ファイルを入れる箱

1. GitHubにログイン
2. 右上の「+」マークをクリック
3. 「New repository」をクリック

### 設定する項目

| 項目 | 入力内容 |
|------|----------|
| Repository name | `wedding-matching` |
| Description | 婚活パーティマッチングシステム |
| Public/Private | **Public**（公開）を選択 |
| Add a README | ☑️ チェックを入れる |

4. 緑色の「Create repository」ボタンをクリック

**完成！** あなた専用の箱ができました。

---

## 2-4. ファイルをアップロードする

1. 「Add file」ボタンをクリック
2. 「Upload files」を選択
3. パソコンからファイルをドラッグ&ドロップ

### アップロードするファイル
- ✅ index.html
- ✅ admin.html
- ✅ css/style.css
- ✅ js/participant.js
- ✅ js/matching.js
- ✅ js/admin.js

### フォルダ構造を守る！
- `css`フォルダを作って、その中に`style.css`
- `js`フォルダを作って、その中にJSファイル3つ

4. 下の方にある緑色の「Commit changes」ボタンをクリック

**完了！** GitHubにファイルが保存されました。
