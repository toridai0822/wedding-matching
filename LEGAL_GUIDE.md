## 1️⃣ 事業形態の選択

### パターンA: 個人事業主（最も簡単）
**メリット**
- ✅ 手続きが簡単（開業届1枚）
- ✅ 初期費用ゼロ
- ✅ すぐに始められる

**デメリット**
- ❌ 個人資産に責任が及ぶ
- ❌ 社会的信用度が低い
- ❌ 大企業との取引が難しい

**必要な手続き**
```
1. 開業届の提出（税務署へ）
   → 事業開始から1ヶ月以内
   → 費用: 無料
   
2. 青色申告承認申請書（任意、推奨）
   → 節税メリット大
   → 費用: 無料
```

### パターンB: 合同会社（バランス型）
**メリット**
- ✅ 個人資産と分離（有限責任）
- ✅ 設立費用が安い（10万円程度）
- ✅ 社会的信用度が高い

**デメリット**
- ❌ 設立に手間がかかる
- ❌ 決算・税務申告が複雑

**必要な手続き**
```
1. 定款作成
2. 資本金の払込（1円〜、推奨50-100万円）
3. 法務局で登記
   → 設立費用: 10万円程度
   → 期間: 2-3週間
```

### パターンC: 株式会社（本格派）
**メリット**
- ✅ 最も信用度が高い
- ✅ 資金調達がしやすい
- ✅ 大企業との取引に有利

**デメリット**
- ❌ 設立費用が高い（25万円程度）
- ❌ 手続きが複雑
- ❌ 維持コストが高い

## 2️⃣ 必須の法的文書（最優先）

### A. 利用規約（Terms of Service）
**必須項目**
```
□ サービス内容の説明
□ 利用条件
□ 禁止事項
□ 料金・支払い条件
□ 免責事項
□ サービス停止・解約条件
□ 知的財産権の帰属
□ 準拠法・管轄裁判所
```

**テンプレート所在**
- 経済産業省「モデル契約書」
- 弁護士ドットコムのテンプレート
- 他社の利用規約を参考（コピペは NG）

### B. プライバシーポリシー（個人情報保護方針）
**必須項目**
```
□ 取得する個人情報の種類
□ 利用目的
□ 第三者提供の有無
□ 保管期間
□ 開示・訂正・削除の方法
□ 問い合わせ窓口
□ クッキーポリシー
```

### C. 特定商取引法に基づく表記
**必須項目**（有料サービスの場合）
```
□ 事業者名（個人の場合は氏名）
□ 所在地（個人の場合は自宅住所でも可）
□ 電話番号
□ メールアドレス
□ 販売価格
□ 支払方法・時期
□ 役務提供時期
□ 返品・キャンセルポリシー
```

**⚠️ 注意**: 個人事業主の場合、自宅住所が公開されます
→ バーチャルオフィス（月額1,000円〜）の利用も検討

## 3️⃣ 個人情報保護対応

### 個人情報保護法の適用
- **5,000件以上**の個人情報を扱う場合 → 個人情報保護法が適用
- 婚活パーティの参加者情報は「個人情報」に該当

### 必要な対応
```
□ プライバシーポリシーの作成・公開
□ 個人情報の利用目的の明示
□ 本人の同意取得（チェックボックス等）
□ セキュリティ対策の実施
  - SSL/TLS通信（HTTPS化）
  - パスワードの暗号化
  - アクセス制限
□ 個人情報管理責任者の設置
□ 漏洩時の対応フロー策定
```

### セキュリティ対策（技術面）
```
優先度1（必須）:
□ HTTPS化（SSL証明書）
□ 管理者パスワードのハッシュ化
□ SQLインジェクション対策
□ XSS対策

優先度2（推奨）:
□ 二要素認証
□ アクセスログの記録
□ 定期的なバックアップ
□ セキュリティ診断の実施
```

## 4️⃣ 契約書類

### A. サービス利用契約書
**BtoB（法人向け）の場合に必要**

**主な内容**
```
□ 契約期間
□ 料金・支払条件
□ サービス内容
□ サービスレベル保証（SLA）
□ 責任範囲・免責事項
□ 秘密保持義務
□ 契約解除条件
□ 損害賠償の上限
```

### B. 秘密保持契約書（NDA）
**企業との商談時に必要**

```
□ 秘密情報の定義
□ 利用目的の制限
□ 開示範囲の制限
□ 返還・破棄義務
□ 有効期間
```

## 5️⃣ 知的財産権

### 商標登録（推奨）
**サービス名を保護**

```
費用: 30,000円〜50,000円
期間: 6ヶ月〜1年
メリット: 他社の模倣を防げる
```

**手続き**
1. 特許庁のサイトで先行商標を調査
2. 商標登録願を提出
3. 審査（6-12ヶ月）
4. 登録査定

### 著作権
**プログラムコードは自動的に著作権保護**
- 特別な手続き不要
- ただし、証拠として残す工夫が必要
  - GitHubでのバージョン管理
  - 公証役場での確定日付

## 6️⃣ 税務・会計

### 個人事業主の場合
```
□ 開業届の提出（税務署）
□ 青色申告承認申請書の提出（推奨）
□ 帳簿の作成・保存義務
□ 確定申告（毎年2-3月）
□ 消費税（年商1,000万円超で課税事業者）
```

**会計ソフトの利用**
- freee: 月額980円〜
- マネーフォワード: 月額800円〜
- やよいの青色申告: 年額8,000円〜

### 法人の場合
```
□ 法人税の申告
□ 消費税の申告
□ 社会保険の加入
□ 法人住民税
□ 税理士への依頼推奨（月額3-5万円）
```

## 7️⃣ その他の検討事項

### A. 損害保険
```
□ 賠償責任保険（サイバーリスク保険）
  - システム障害による損害をカバー
  - 年額5万円〜
  
□ 事業総合保険
  - 事業活動全般のリスクをカバー
```

### B. ドメイン・商号
```
□ 独自ドメインの取得
  - .com: 年額1,500円〜
  - .jp: 年額3,000円〜
  
□ 商号（会社名・屋号）の決定
  - 同業他社と類似していないか確認
```

## ⚠️ よくある失敗・注意点

### 失敗1: 利用規約なしで販売開始
❌ トラブル時に責任範囲が不明確
✅ 必ず利用規約を作成・同意取得

### 失敗2: 個人情報の取り扱いが不適切
❌ 漏洩時に大問題、信用失墜
✅ セキュリティ対策を最優先

### 失敗3: 税務処理を怠る
❌ 無申告加算税・延滞税のペナルティ
✅ 会計ソフトで日々記帳

### 失敗4: 契約書なしで大口取引
❌ 支払いトラブル、責任問題
✅ 必ず契約書を締結
